Artify Backend

This is the backend server for Artify, a full-stack art sharing platform where users can create, view, like, and favorite artworks. The backend is built with Node.js, Express.js, MongoDB, and Firebase Authentication for secure user management.

Features

User authentication and authorization using Firebase Admin SDK

CRUD operations for artworks

Artwork ownership verification for updates and deletions

Favorite and like management for users

Secure routes protected by JWT (Firebase tokens)

Search artworks by title or artist name

Support for light/dark mode through frontend integration

Technologies Used

Node.js – Server runtime

Express.js – Backend framework

MongoDB – Database

Firebase Admin SDK – Authentication

Cors – Middleware for handling cross-origin requests

dotenv – Environment variable management
